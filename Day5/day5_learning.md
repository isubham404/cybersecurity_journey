# Day 5 - Encryption & Decryption

## What I Learned

### Encryption
- Converts readable text into unreadable format
- Uses a key (shift)

### Decryption
- Converts encrypted text back to original
- Uses reverse operation

### Key Idea
- Encryption = + shift
- Decryption = - shift

## What I Built
- A Caesar Cipher tool
