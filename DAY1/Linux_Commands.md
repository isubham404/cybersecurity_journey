🐧 Essential Linux Terminal Commands

📁 <b><i>File & Directory Navigation</i></b>

<b>Command Description</b>

<b>ls</b> --> List directory contents
<b>ls -la</b> --> List all files (including hidden) with detailed permissions
<b>cd <dir></b> --> Change directory
<b>pwd</b> --> Print working directory (where am I?)
<b>mkdir <name></b> --> Create a new directory
<b>rm -rf <file/dir></b> --> Forcefully remove a file or directory (use with caution!)
<b>cp -r <src> <dest></b> --> Copy files or directories recursively
<b>mv <src> <dest> </b> --> Move or rename files/directories

<b><u> 🛠️ System & Permissions </u> </b>

</b>Command Description</b>

<b> sudo <command> </b> --> Execute a command with root privileges
<b>chmod +x <file> </b> --> Make a script/file executable
<b>chown <user>:<group></b> --> Change file owner and group
<b>top / htop </b>  --> Display active system processes
<b>ps aux </b> --> Snapshot of all running processes
<b>kill -9 <pid> </b>  --> Forcefully terminate a process by ID
<b>clear</b> --> Clear the terminal screen


<b><u> 🌐 Networking & Connectivity </u></b>

<b>Command Description</b> 

<b>ifconfig / ip a </b> --> View network interfaces and IP addresses
<b>iwconfig </b>--> View wireless network interfaces
<b>ping <host> </b>--> Check connectivity to a host
<b>netstat -tunlp</b> --> View active ports and listening services
<b>ssh user@host </b>--> Secure Shell remote access
<b>wget <url> </b>--> Download files from the web via terminal
<b>curl -I <url></b> --> Fetch HTTP headers from a URL
