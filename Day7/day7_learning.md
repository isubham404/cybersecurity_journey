# Day 7 - Password Verification using Hashing

## Objective
To implement a simple password verification system using hashing.

## Concept
Passwords should not be stored in plain text. Instead, their hash values are stored.

## Process
1. Store hashed password
2. Take user input
3. Hash user input
4. Compare hashes

## What I Learned
- Real-world use of hashing
- Authentication process
- Secure password handling basics
